# Courses — App PWA

Cette version repart de la v67 avec la data intégrée par défaut.

## Tester sur téléphone

1. Dézippe le dossier.
2. Héberge le dossier avec un petit serveur local ou en ligne.
   Exemple rapide depuis le dossier :
   `python3 -m http.server 8000`
3. Ouvre l’URL sur ton téléphone.
4. Sur iPhone : Partager > Sur l’écran d’accueil.
5. Sur Android/Chrome : menu ⋮ > Installer l’application.

Important : l’installation PWA nécessite une URL servie en HTTP/HTTPS, pas une ouverture directe du fichier local.
